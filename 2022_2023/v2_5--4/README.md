This robot features:

## Configuration guide
### Motors:
 - frontright : port 0 CH
 - backleft : port 1 CH
 - frontleft : port 2 CH
 - backright : port 3 CH
 - horizontalslide : port 0 EH
 - linearslide : port 1 EH

### Servos:
 - ring : port 0 CH
 - raisearm : port 1 CH
 - griparm : port 1 CH
